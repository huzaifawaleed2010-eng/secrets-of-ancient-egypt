# 🏛️ أسرار مصر القديمة (Secrets of Ancient Egypt)

موقع ويب تفاعلي وتعليمي يأخذ المستخدم في رحلة استكشافية عبر الزمن ليتعرف على تاريخ الحضارة المصرية القديمة، ملوكها، وأبرز عصورها التاريخية، بالإضافة إلى توفير أداة تفاعلية مميزة.

---

## ✨ مميزات الموقع (Features)

* **تصميم متجاوب بالكامل (Fully Responsive):** الموقع متوافق مع جميع الشاشات (موبايل، تابلت، كمبيوتر) بفضل استخدام الـ CSS Media Queries.
* **تصفح العصور التاريخية:** تقسيم برمجى منظم للمحتوى يبدأ من العصر العتيق وحتى العصر المتأخر.
* **مترجم الأسماء للهيروغليفية:** أداة تفاعلية مبرمجة بـ JavaScript تتيح للمستخدم كتابة اسمه وتحويله إلى الرموز المصرية القديمة.
* **شاشة تحميل احترافية (Splash Screen):** تجربة مستخدم سلسة تظهر أثناء تحميل ملفات الموقع والصور.

---

## 🛠️ التقنيات المستخدمة (Tech Stack)

تم بناء هذا المشروع باستخدام التقنيات الأساسية لتطوير الويب (Front-end):

* **HTML5:** لبناء الهيكل البرمجي للصفحات وتهيئتها لمحركات البحث (SEO).
* **CSS3:** لتنسيق الألوان، الخطوط، وتأثيرات الحركة (Animations) والـ Flexbox/Grid.
* **JavaScript (ES6):** لبرمجة المنطق التفاعلي (Interactive Logic) الخاص بمحول اللغة والقوائم المتجاوبة.
